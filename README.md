# Connect Four

## Installation
1. Cd into the project folder
   
![CD into folder](/images_readme/cd.png)
   

2. Install pygame

```
pip3 install pygame
or
pip install pygame
```

3. Install tkinter

```
pip3 install tkinter
or
pip install tkinter
```

4. Run the project
```
python3 connect_four.py
or
python connect_four.py
```

![Enter options](/images_readme/options.png)

5. Play the game
